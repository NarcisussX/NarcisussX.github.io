Conference-GO

Project Info:

Conference-GO is a feature heavy website designed to manage upcoming conferences and attendees. Conference-GO uses third party API calls for conference location weather data from a third party API, RabbitMQ mailing for approved conference presentation notices, user authentication and account registration.
