Project Info:

CampSight is a lightweight and fast web project designed to help a user search for and navigate all the National Parks. It features two FastAPI services hosted on Render, a MongoDB database and a front-end made in React. Features include geolocation, search filtering, user account creation, reviews, event planning and favoriting visited parks.
